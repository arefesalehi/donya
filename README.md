# 🎓 CodeCourses - Programming Courses Marketplace

A modern full-stack **online learning platform** for programming courses, built with **React**, **custom backend API**, and **Tailwind CSS**, featuring a complete **User & Admin Dashboard**, ticketing system, and course management system.

## ✨ Key Features

### 👤 User Panel
- Secure Authentication (Login / Register)
- Browse programming courses
- Purchase / enroll in courses
- Track enrolled courses
- Submit support tickets
- Profile management
- Responsive UI

---

### 🛡️ Admin Panel
- Full course management (Create / Edit / Delete)
- Manage users & roles
- Ticket management system
- Reply to user support tickets
- Dashboard analytics
- Content moderation

---

### 🎓 Course System
- Create structured programming courses
- Add lessons & modules
- Video / content management support
- Category-based filtering
- Search functionality

---

### 🎫 Ticket System
- Users can submit support tickets
- Admin can reply & manage tickets
- Real-time status tracking (open / pending / closed)
